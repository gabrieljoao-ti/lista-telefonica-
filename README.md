# lista-telefonica-
Sistema de Gerenciamento de Contatos

Este projeto consiste em um sistema de gerenciamento de contatos desenvolvido em linguagem C, utilizando estruturas (struct), vetores e funções para organizar e manipular informações de usuários.

Atualmente, o sistema permite o cadastro de contatos por meio de um menu interativo no terminal. Cada contato possui os seguintes dados:

Nome
E-mail
Telefone
Tipo de contato (Pessoal, Trabalho, Família, Cliente ou Outro)

Os registros são armazenados em memória utilizando um vetor de estruturas com capacidade para até 150 contatos.

Funcionalidades implementadas:
Cadastro de novos contatos
Validação de limite máximo de armazenamento
Menu interativo de navegação
Organização dos dados por meio de estruturas (struct)
Tratamento de entrada de texto utilizando fgets()
Funcionalidades planejadas
Alteração de contatos existentes
Exclusão de contatos
Busca de contatos
Listagem completa dos registros
Persistência de dados em arquivo (contatos.dat)
Tecnologias utilizadas
Linguagem C

Bibliotecas:
stdio.h
stdlib.h
string.h

Conceitos praticados:
Estruturas (struct)
Vetores de registros
Passagem de parâmetros por referência
Modularização com funções
Manipulação de strings
Menus interativos
Controle de fluxo com switch-case
Gerenciamento básico de dados

Objetivo do projeto:
O objetivo deste projeto é praticar conceitos fundamentais da programação em C, simulando uma agenda de contatos capaz de armazenar, consultar e gerenciar informações de forma organizada. O sistema está sendo desenvolvido de maneira incremental, com novas funcionalidades sendo adicionadas conforme o avanço dos estudos.
